# Orange cat doc

It is an a markup language that is like markdown but has more commands and markup rules

Theres are the 'pseudotags':

- !: Title of the page
- $: H1 title
- $\$: H2 title
- **: Strong text
- /: Cursive text
- ~: strikethrough text
- _: underlined text

## Example page with odoc

```odoc
! Example page

$ My h1 title
$$ My h2 title

** strong text
/ cursive text
~ srikethrough text
_ underlined text
```
